# PC音频同步软件

## 功能
- 通过网络将电脑的音频流转到其他设备进行播放
- 可以自由指定音频设备，也可以跟随系统默认值
- 可以同时流转多路音频，也就是可以将多台电脑的音频在一台电脑播放
- 支持指定来源和播放的声卡设备
- 支持从本机音频设备到另外一个音频设备
- 可以实现多台电脑播放电脑本身的声音外同时播放其他电脑的声音
- 极低延迟

## 注意
- 音频数据在传输时没有经过压缩，占用3M的带宽（千兆网无压力，百兆网如果同时播放的太多网络压力较大）
- 没有鉴权，输入ip就能被获取音频设备列表和拉取音频，请勿在不安全的网络使用

## 使用步骤
### 安装
1. 下载软件
2. 解压到你需要安装的目录
3. 双击打开AudioStream.exe
4. 需要流转音频的两台电脑都要安装
### 使用
1. 在任务栏托盘找到应用右键或双击打开配置界面
2. 输入电脑ip，选择来源的音频设备和播放的音频设备后添加即可
3. 如果不输入ip表示流转的是当前电脑的音频
